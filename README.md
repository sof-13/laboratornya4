# laboratornya4
Работа представляет собой портирование консольного приложения на Java в десктопное приложение с графическим интерфейсом на основе Swing.

Программа позволяет добавлять типы работ с базовой оплатой или надбавкой, просматривать список и рассчитывать среднюю зарплату. В графической версии эта логика сохранена полностью и дополнена удобным интерфейсом: данные отображаются в таблице, поддерживаются добавление, редактирование и удаление записей, ввод вручную, валидация полей, автоматическая сортировка по названию, а также возможность сохранять и загружать данные из текстового файла. Приложение состоит из двух окон, соответствует всем требованиям задания и может быть собрано в исполняемый JAR-файл.
